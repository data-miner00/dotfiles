# dotfiles

The not-so-fancy configurations that I use for my second Arch Linux laptop.

## Stacks

- dwm - window manager
- dwmblocks - system-tray for dwm
- Firefox - browser
- pipewire - audio
- picom - window transparency and animations
- neovim - text editor
